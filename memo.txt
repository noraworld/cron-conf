ファイルのパスを指定しない場合はホームディレクトリになる
逆を言うと絶対パスを指定すれば任意のパスに書き込むことができる

sudo が必要なジョブを実行したいときは root としてジョブを登録する
一般ユーザで sudo を無理やり実行するのは危険だしとても面倒くさすぎる

cron ジョブは crontab -e に書き込むことで設定できるが推奨されない
cron.conf にジョブを書き込んでそれを crontab に反映させるほうが良い

cron.conf を crontab に反映させるためには以下のコマンドを実行する
$ crontab      < cron.conf       # 一般ユーザの場合
$ sudo crontab < root_cron.conf  # root ユーザの場合

crontab に反映されているジョブを確認するには以下のコマンドを実行する
$ crontab -l       # 一般ユーザの場合
$ sudo crontab -l  # root ユーザの場合
